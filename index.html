About Graphaxe

Graphaxe is an advanced, high-performance 2D game engine crafted with Haxe. Combining lightweight architecture with a rich set of features, Graphaxe empowers developers to create visually stunning and highly optimized games with unparalleled ease and efficiency. Whether you're an indie developer or part of a large studio, Graphaxe provides the tools and flexibility needed to bring your creative visions to life, all while maintaining the speed and simplicity that modern game development demands.
